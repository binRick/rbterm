# Bundled fonts

rbterm bundles a small set of open-source monospace fonts so the
settings font-picker has sensible choices out of the box. Each of
these comes from the upstream project under a permissive licence —
see the linked repos for the full licence text.

| Font              | Upstream                                             | Licence               |
|-------------------|------------------------------------------------------|-----------------------|
| JetBrains Mono    | https://github.com/JetBrains/JetBrainsMono           | Apache-2.0            |
| Fira Code         | https://github.com/tonsky/FiraCode                   | OFL-1.1               |
| Source Code Pro   | https://github.com/adobe-fonts/source-code-pro       | OFL-1.1               |
| Hack              | https://github.com/source-foundry/Hack               | MIT + Bitstream Vera  |
| Inconsolata       | https://github.com/googlefonts/Inconsolata           | OFL-1.1               |
| IBM Plex Mono     | https://github.com/IBM/plex                          | OFL-1.1               |

None of the bundled files have been modified from upstream. If you
redistribute rbterm you need to preserve the licence notices.
